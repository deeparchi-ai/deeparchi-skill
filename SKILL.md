# DeepArchi Skill - ArchiMate 架构建模

## 概述

DeepArchi Skill 是一个专门用于创建符合 ArchiMate 3.2 标准的企业架构图的技能。该技能集成了 draw.io 工具，帮助架构师快速创建标准化的企业架构视图。

## 何时使用

当用户需要：
- 创建符合 ArchiMate 标准的企业架构图
- 使用 draw.io 绘制架构视图
- 生成业务架构、应用架构、技术架构等视图
- 创建架构资产的可视化表达
- 遵循 TOGAF 和 ArchiMate 标准的架构设计

## 快速开始

1. 在 draw.io 中启用 ArchiMate 3.2 形状库
2. 参考 `references/drawio-archimate-tutorial.md` 学习基本操作
3. 查看 `EXAMPLES.md` 了解实际应用场景
4. 使用 `scripts/validate-archimate.js` 验证图表符合性

## 核心功能

- **ArchiMate 3.2 标准支持**：完整支持业务层、应用层、技术层元素
- **关系类型指导**：11 种标准关系类型的使用指南
- **颜色规范**：符合 ArchiMate 标准的颜色调色板
- **验证工具**：JavaScript 脚本验证图表符合性
- **示例模板**：多个实际应用场景示例
